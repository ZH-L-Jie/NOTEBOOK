## **三、快捷键**

###1. 快速定位到方法的代码块的结尾或开始快捷键

- CTRL+shift + { ：快速定位到代码块的开始
- CTRL+shift + } ：快速定位到代码块结尾
- Ctrl + Alt + 鼠标左键 ： 进入该方法的实现，而不是接口
- Ctrl+Y ：快速删除一行
